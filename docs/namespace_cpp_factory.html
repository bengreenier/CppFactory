enum CppFactory::ObjectLifecycle

Represents the lifecycle of an instance

We use this value to influence how internal state is tracked for object re-use (Global) and knowing when to destruct instances

Enumerator
Untracked 

The object should be untracked, subsequent calls to Object<TObject, TObjLifecycle>.Get(int zone) will result in a new instance

Global 

The object should be tracked and tied to the global lifecycle subsequent calls to Object<TObject, TObjLifecycle>.Get(int zone) will reuse the instance
